# Book Catalog

**Проект:** Каталог книг и авторов  

**Описание:**  
Book Catalog — это веб-приложение на Django для хранения информации о книгах, их авторах и деталях произведений. Приложение позволяет:  
- Просматривать авторов и их биографии  
- Смотреть список книг и краткие описания  
- Хранить дополнительные детали для каждой книги  

**Технологии:**  
- Python 3.11  
- Django 5.2  
- PostgreSQL  

**Особенности:**  
- Использует реляционную базу данных PostgreSQL  
- Легко расширяемая структура моделей: Author, Book, BookDetail  
- Административная панель для управления данными  

**Для кого:**  
- Для разработчиков, которые хотят изучать Django с реальным примером  
- Для любителей книг, желающих вести личный каталог
